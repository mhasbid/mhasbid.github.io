HAI

Perkenalkan nama saya Muhammad Hasbi Damaryana, bisa dipanggil hasbi atau damar. Saya lahir tahun 2004 tepatnya pada 18 Juli di Bogor. Saya merupakan seorang siswa di salah satu sekolah kejuruan ternama di bogor yaitu SMK SMAK Bogor. Saat ini saya sedang menimba ilmu di tahun ke 2.

Disini saya akan membagikan pengalaman saya selama pembelajaran jarak jauh. Pembelajaran jauh saat pandemi memberikan saya banyak pembelajaran bagaimana caranya menghadapi situasi seperti ini, memang pada awal pandemi saya kelelahan dalam belajar, namun seiring berjalannya waktu saya menemukan berbagai metode agar belajar menjadi asyik dan menyenangkan. Salah satu metode belajar saya yaitu dengan memberikan hadiah terhadap diri sendiri, misal jika saya selesai mengerjakan tugas ini maka saya boleh bermain game atau mungkin boleh jalan- jalan. Hal itu sangat efektif untuk memberikan peningkatan semangat belajar saya.

Disini juga saya akan membagikan beberapa foto mengenai hobi saya, yaitu olahraga, bermain alat musik dan travelling
